OVERVIEW MAP OF SINGAPORE



Guide

Button What it does
Zoom ( '+' , '-') Adjust zoom level of the map. Double clicking a certain point in the map will also zoom in to that point
Map Type control (Map Satelite button) A dropdown or horizontal button bar style at top left of map, allowing the user to choose a map type (ROADMAP, SATELLITE, HYBRID, or TERRAIN).
Street View control (human icon) Human icon which can be dragged onto the map to enable Street View.
Fullscreen control Able to open the map in fullscreen mode.
Note: iOS doesn't support the fullscreen feature. The fullscreen control is therefore not visible on iOS devices.
'Where am I?' Pinpoint user's current location
